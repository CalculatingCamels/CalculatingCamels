How To Use Treadstone

Treadstone is the premiere bike route sharing web application on the internet! Search for your current location on the main page to preview routes that other bikers have already written, or create a profile to add your own routes.

About Calculating Camels

The Calculating Camels is a patchwork group of coding mercenaries whose mother's are always telling them how smart they are. Jamie Skinner did the styling and maps, Tyler Lambe built the back-end, Christian Brandalise and Charlie McBride built the front-end and maps, and Brian Small built the testing suite.
